عرض أيقونة مخصصة للعملة بدلاً من الرمز النصي العادي في متجر WooCommerce

== Description ==

**Custom Currency Icon for WooCommerce** يسمح لك بـ:

- رفع صور مخصصة لكل عملة (PNG, JPG, WebP)
- اختيار طرق عرض متعددة (قبل السعر، بعده، صورة فقط، إلخ)
- التحكم الكامل في حجم ومحاذاة الأيقونة
- عرض الأيقونة في جميع أماكن الأسعار (منتجات، سلة، دفع، طلبات)
- معاينة مباشرة للتغييرات
- Shortcode للعرض المخصص
- تحديثات تلقائية من GitHub

**المميزات:**
✓ واجهة سهلة الاستخدام
✓ معاينة مباشرة
✓ أمان عالي (CSRF, Nonce, Sanitization)
✓ دعم WooCommerce كامل
✓ Responsive Design
✓ دعم التحديث من GitHub
✓ عدم الاعتماد على external libraries

== Installation ==

1. قم بتحميل ملفات البلاجن إلى `/wp-content/plugins/custom-currency-icon-for-woocommerce/`
2. فعّل البلاجن من قسم Plugins
3. اذهب إلى WooCommerce > Currency Icon
4. رفّع صور عملاتك واختر الإعدادات المطلوبة

== Frequently Asked Questions ==

= هل يدعم البلاجن عملات متعددة؟ =

نعم! يمكنك رفع صورة مخصصة لكل عملة في متجرك.

= هل يعمل مع متاجر WooCommerce متعددة العملات؟ =

نعم، يدعم جميع العملات التي تدعمها WooCommerce.

= كيف أرفع صورة جديدة؟ =

1. اذهب إلى WooCommerce > Currency Icon
2. اختر العملة من القائمة
3. انقر "Upload Image" واختر صورة

= هل الصور آمنة؟ =

نعم تماماً! يتم فحص كل الملفات المرفوعة وتحديد نوعها قبل الحفظ.

= هل يؤثر على الأداء؟ =

لا، البلاجن خفيف جداً ولا يؤثر على سرعة موقعك.

= كيف أحصل على التحديثات التلقائية؟ =

البلاجن يدعم التحديث المباشر من GitHub. عند نشر release جديد، ستراه في لوحة التحكم.
